Šablona

Moderní, responzivní a stylová stránka pro GitHub Pages

Připravená šablona se sekcemi, animacemi, responsivním layoutem a jednoduchým JavaScriptem pro menu a přepínač vzhledu (dark/light). Stačí nahrát index.html do repozitáře a aktivovat GitHub Pages.

Rychlé nasazení

Nahrát soubor index.html do větve main (nebo gh-pages) a zapnout GitHub Pages.

Responzivní

Grid a media queries zajišťují skvělý vzhled na mobilech i desktopu.

Snadná úprava

Barvy, písmo a obsah upravíš přímo v tomto souboru. Komentáře v kódu pomohou.
